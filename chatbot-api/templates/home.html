HelpTree-Chatbot API

Parameter yang digunakan -> query (isi dengan pesan yang ingin dikirimkan)

Cara menggunakan (langsung menggunakan browser): http://rezkaz.pythonanywhere.com/bot?query=[tulis pesan disini tanpa menggunakan kurung siku] (klik disini untuk contoh)

Note untuk android-dev: Kalo get data json dari Android, coba cari referensi lagi ham mut(kalo ga salah json decode gitu)


Hasil (bentuk JSON):

    user_query -> input/query yang dikirimkan
    prediksi_intent -> prediksi intent/class dari input (untuk menentukan respons)
    confidence -> bisa dibilang akurasi dari prediksi, makin besar makin bagus
    response -> respon/jawaban dari bot yang akan dikirimkan ke user (cukup tampilkan yang ini aja di chatbot)


Untuk saat ini hanya bisa memprediksi 3 kelas intent, yaitu:

    rekomendasi_tanaman (contoh query-> saya ingin menanam tanaman)
    salam_pembuka (contoh query-> hai, halo)
    selesai (contoh query-> terima kasih)
